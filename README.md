# agent-skills

Agent skills for Claude Code, Codex, and other AI coding agents.

## Install

All skills:

```
npx skills add sudorock/agent-skills
```

Single skill:

```
npx skills add sudorock/agent-skills --skill terse
```

## Skills

| Skill | Description |
|---|---|
| [terse](terse/) | Structured, information-dense output with coordinate addressing. ~75% token reduction without information loss. Every line gets a stable address for referencing, expanding, and superseding. |
